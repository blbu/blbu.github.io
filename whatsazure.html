What's Azure?

Disclaimer: I know very little about Azure. This is probably misinformation.

Azure is "the cloud" from Microsoft. That means we can get VMs and all sorts of other good stuff. Remember, the cloud is just someone else's computer—but if you're happy to have them patch it for you and so on, great! Utilziing technology like Azure let's you defer up-front capital investment into your servers, and "scale-out" more rapidly—by leveraging their server capacity.

A basic use case (IaaS): you pay for a virtual machine running one of thousands of pre-built images, and you do whatever you want with it. This is easy to get into, because it's barely more complicated than "VirtualBox on steroids". One model might be fail-over/redundancy for on-premise servers. Another could be "lift and shift"/"P2V"—virtualizing "box under desk" servers but not utilizing much more than that.

Another use case (PaaS): you buy into the "platform" offerings, such as high-performance, distributed queueing, micro-functions which run on-demand and so on. This is harder to get into, because you have to do things "the cloud way". Models might be a cloud-only startup company.
